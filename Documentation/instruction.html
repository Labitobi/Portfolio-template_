1. Requirements

  • Node.js 18+ and npm installed
  • Basic understanding of Next.js file structure

2. Installation

  1. Open your terminal inside the theme/ folder.
  2. Run npm install --legacy-peer-deps to install dependencies.
  3. Start the development server: npm run dev

Note: The --legacy-peer-deps flag resolves minor compatibility issues.

3. Editing the Template

  • Pages are located inside app/ directory.
  • Components are inside components/.
  • Animations use Framer Motion inside component files.
  • Global styles are managed in styles/globals.css.

4. Customizing Logo, Colors, and Images

  • Replace the logo image in public/logo.svg.
  • Edit color themes inside tailwind.config.js.
  • Change project images in public/Project-images/ folder.

5. Building for Production

  • Run npm run build to create an optimized production build.
  • Deploy to Vercel, Netlify, or any modern hosting platform easily.
