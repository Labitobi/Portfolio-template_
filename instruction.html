1. Requirements

  • Node.js and npm installed (recommended version: Node 18+)
  • Basic understanding of Next.js structure
  • Framer Motion is used for page and element animations

2. Installation

  1. Open your terminal in the project folder.
  2. Run npm install --legacy-peer-deps to install dependencies.
  3. Start the dev server with npm run dev.

Note: This flag skips peerâ€‘dependency conflicts and is safe to use with this template.

3. Editing the Content

  • Pages are located in src/pages/.
  • Components are located in src/components/.
  • Animations are powered by Framer Motion inside these components.
  • Global styles are managed in src/styles/globals.css.
  • Update text, images, and project data inside the relevant files.

4. Building for Production

  • Run npm run build to create an optimized production build.
  • Deploy easily to Vercel, Netlify, or any modern hosting platform.
